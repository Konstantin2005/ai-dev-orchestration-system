# Agent System

## Roles

### 🧭 Architect
- **Directory:** `00-architect/`
- **Files:** `plan.md`, `architecture.md`, `decisions.md`
- **Task:** System design, architecture alternatives, trade-offs

### ⚙️ Backend Engineer
- **Directory:** `01-backend-engineer/`
- **Files:** `api.md`, `implementation.md`, `data-model.md`
- **Task:** API, business logic, data model

### 🎨 Frontend Engineer
- **Directory:** `02-frontend-engineer/`
- **Files:** `ui.md`, `components.md`, `state-model.md`
- **Task:** UI, components, state management

### 🧪 QA Engineer
- **Directory:** `03-qa-engineer/`
- **Files:** `tests.md`, `edge-cases.md`, `coverage.md`
- **Task:** Test cases, edge cases, stress scenarios

### 🔍 Code Reviewer
- **Directory:** `04-code-reviewer/`
- **Files:** `review.md`, `risks.md`, `security.md`
- **Task:** Security audit, quality check, final verdict

## Rules
- Each agent writes ONLY in its directory
- Communication via `/workspace/issues/<id>/shared/context.md`
- No cross-agent file edits
