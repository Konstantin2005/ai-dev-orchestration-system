# Роли агентов

## 🧭 Architect
- Проектирует систему
- Разбивает задачу на подзадачи
- Определяет архитектуру
- НЕ пишет production code
- **Папка:** `issues/<id>-<slug>/00-architect/`

## ⚙️ Backend Engineer
- API, бизнес-логика, база данных, интеграции
- **Папка:** `issues/<id>-<slug>/01-backend-engineer/`

## 🎨 Frontend Engineer
- UI, компоненты, состояние приложения, UX логика
- **Папка:** `issues/<id>-<slug>/02-frontend-engineer/`

## 🧪 QA Engineer
- Тест-кейсы, edge cases, регрессия, стабильность
- **Папка:** `issues/<id>-<slug>/03-qa-engineer/`

## 🔍 Code Reviewer
- Безопасность, качество кода, архитектурные ошибки, best practices
- **Папка:** `issues/<id>-<slug>/04-code-reviewer/`
