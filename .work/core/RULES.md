# Правила системы

1. **Изоляция** — каждый агент работает ТОЛЬКО в своей папке
2. **Запрет перекрёстных правок** — нельзя редактировать чужие папки
3. **Коммуникация** — только через `shared/`
4. **Все решения фиксируются в файлах**, не в чатах и GitHub comments
5. **Никакой работы вне `.work/issues/`**
6. **Приоритет — `shared/architecture.md`** — архитектурные изменения сначала туда
